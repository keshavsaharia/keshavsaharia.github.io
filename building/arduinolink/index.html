ArduinoLink

During my employment at Wolfram Research in the Kernel Technology department, I had the opportunity to create new use cases for the Mathematica programming language.

ArduinoLink is an open-source package for using the Mathematica language with the Arduino microcontroller. Below is a talk that I gave with ArduinoLink's biggest fan, Christopher Wolfram (son of Stephen Wolfram, my employer at that time). This was at the Wolfram Technology Conference in 2012.

The projects and prototypes that I created were used by Stephen and his son in their talk “Computation Everywhere” at the World Maker Faire in 2012.

You can find the source for ArduinoLink on GitHub. It only supports MacOSX, but I would love to see it as a production-ready package that can support multiple operating systems. If you have ideas or comments on how to go about doing this, please leave an issue on the GitHub page and I will try to respond as quickly as possible. Happy prototyping!
